## Problem

> **문제**
  ```
  > OX 문제는 맞거나 틀린 두 경우의 답을 가지는 문제를 말한다.
  > 여러 개의 OX 문제로 만들어진 시험에서 연속적으로 답을 맞히는 경우에는 가산점을 주기 위해서 다음과 같은 점수 계산을 하기로 했다.
  > 1번 문제가 맞는 경우에는 1점으로 계산하고, 1번 문제 이후로 답을 틀리다가 답이 맞은 처음 문제는 1점으로 계산한다.
  > 또한, 연속으로 문제의 답이 맞는 경우에는 두 번째 문제를 2점, 세 번째 문제를 3점, ... , K번째 문제를 K점으로 계산한다.
  > 반면, 틀린 문제는 0점으로 계산한다.
  > 시험문제의 채점 결과가 주어졌을 때, 총 점수를 계산하는 프로그램을 작성하시오.

  > 예를 들어, 아래와 같이 10개의 OX 문제에서 답이 맞은 문제의 경우에는 1로 표시하고,
  > 틀린 경우에는 0으로 표시했을 때 점수 계산은 1+0+1+2+3+0+0+1+2+0이 되어 총 점수는 10점이다.

                                            1 0 1 1 1 0 0 1 1 0
  
  - 입력설명
    * 첫째 줄에 문제의 개수 N(1 ≤ N ≤ 100)이 주어진다.
    * 둘째 줄에는 N개 문제의 채점 결과를 나타내는 0 혹은 1이 빈 칸을 사이에 두고 주어진다.
    * 참고로, 0은 문제의 답이 틀린 경우이고, 1은 문제의 답이 맞은 경우이다.
  
  - 출력설명
    * 첫째 줄의 입력으로 주어진 채점 결과에 대하여, 가산점을 고려한 총 점수를 출력한다. 
  ```

<br>
<hr>

## Solution

> **입력**
  ```
  10
  1 0 1 1 1 0 0 1 1 0
  ```

<br>
<br>

> **풀이**
  ```python
  import sys
  sys.stdin=open('input.txt', 'rt')

  n = int(input())
  ch = list(map(int, input().split()))

  sum = 0
  cnt = 0

  for x in ch:
      if x == 1:
          cnt += 1
          sum += cnt
      else:
          cnt = 0
  
  print(sum)

  """
  출력: 10
  """
  ```

<br>
<br>

> **설명**
  ```
  1. 변수 cnt를 점수 가산점 값으로 설정하고, 문제를 맞추는 경우에만 cnt의 값을 1씩 증가시킵니다.
  2. 즉, 연속으로 문제를 맞추는 경우에만 변수 sum에 변수 cnt를 더하여 가산점 값을 누적합니다.(틀리는 경우에는 cnt를 0으로 초기화)
  ```
