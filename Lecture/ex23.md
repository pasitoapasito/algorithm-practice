## Problem

> **문제**
  ```
  > 임의의 N개의 숫자가 입력으로 주어진다.
  > N개의 수를 오름차순으로 정렬한 다음 N개의 수 중 한 개의 수인 M이 주어지면,
  > 이분 검색으로 M이 정렬된 상태에서 몇 번째에 있는지 구하는 프로그램을 작성하시오.
  > 단, 중복값은 존재하지 않는다.

  - 입력설명
    * 첫 번째 줄에 자연수 N(3 ≤ N ≤ 1,000,000)과 M이 주어진다.
    * 두 번째 줄에 N개의 수가 공백을 사이에 두고 주어진다.

  - 출력설명
    * 첫 줄에 정렬 후 M값의 위치를 번호로 출력한다. 
  ```

<br>
<hr>

## Solution

> **입력**
  ```
  8 32
  23 87 65 12 57 32 99 81
  ```

<br>
<br>

> **풀이**
  ```python
  import sys
  sys.stdin=open('input.txt', 'rt')

  n, m = map(int, input().split())
  ch = list(map(int, input().split()))

  ch.sort()
  lt = 0
  rt = n-1

  while lt <= rt:
      mid = (lt + rt) // 2
      if ch[mid] == m:
          print(mid + 1)
          break
      elif ch[mid] > m:
          rt = mid - 1
      else:
          lt = mid + 1
  
  """
  출력: 3
  """
  ```

<br>
<br>

> **설명**
  ```
  1. 이분검색은 밑이 2인 logn번 이내로 총 n개의 데이터 중 특정 값을 찾는 검색 방법입니다.
  2. 이분검색을 하기 위해서는 n개의 데이터가 오름차순으로 정렬되어 있어야 합니다.
  3. 포인트 변수 lt와 rt를 활용하여 두 변수의 중앙값(mid)을 기준으로 범위를 줄여나가면서, mid가 원하는 값(m)에 도달할 때까지 검색을 반복합니다.
  4. 예를 들어, mid > m인 경우라면 포인트 변수 rt를 mid-1 자리로 옮겨서 이분검색을 다시 진행합니다.
  5. 반대로, mid < m인 경우라면 포인트 변수 lt를 mid+1자리로 옮겨서 이분검색을 다시 진행합니다.
  6. 참고로, 변수 lt, rt, mid는 모두 리스트 ch의 인덱스 위치를 의미합니다.
  ```
