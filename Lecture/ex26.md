## Problem

> **문제**
  ```
  > N개의 마구간이 수직선상에 있다. 
  > 각 마구간은 x1, x2, x3, ... , xN의 자표를 가지며, 마구간의 좌표가 중복되는 일은 없다.
  > A는 m마리의 말을 가지고 있는데, 이 말들은 서로 가까이 있는 것을 싫어한다.
  > 각 마구간에는 한 마리의 말을 넣을 수 있고, 가장 가까운 두 말의 거리가 최대가 되도록 마구간에 말을 배치하고 싶다.
  > m마리의 말을 N개의 마구간에 배치했을 때, 가장 가까운 두 말의 거리가 최대가 되며 그 최대 값을 출력하는 프로그램을 작성하시오.

  - 입력설명
    * 첫 번째 줄에 자연수 N(3 ≤ N ≤ 200,000)과 m(2 ≤ m ≤ N)가 공백을 사이에 두고 주어진다.
    * 두 번째 줄부터 N개의 줄에 걸쳐 마구간의 좌표 xi(0 ≤ xi ≤ 1,000,000,000)가 한 줄에 하나씩 주어진다.

  - 출력설명
    * 첫 번째 줄에 가장 가까운 두 말의 최대 거리를 출력한다.
  ```

<br>
<hr>

## Solution

> **입력**
  ```
  5 3 
  1
  2
  8 
  4   
  9
  ```

<br>
<br>

> **풀이**
  ```python
  import sys
  sys.stdin=open('input.txt', 'rt')

  def count(len):
      cnt = 1
      ep = line[0]
      for i in range(1, n):
          if line[i] - ep >= len:
              cnt += 1
              ep = line[i]
      return cnt
  
  if __name__ == '__main__':
      n, m = map(int, input().split())
      line = []
      for _ in range(n):
          tmp = int(input())
          line.append(tmp)
      line.sort()

      lt = 1
      rt = line[n-1]

      while lt <= rt:
          mid = (lt + rt) // 2
          if count(mid) >= m:
              res = mid
              lt = mid + 1
          else:
              rt = mid - 1
      
      print(res)
  
  """
  출력: 3
  """
  ```

<br>
<br>

> **설명**
  ```
  1. 이분 검색은 특정 범위안에 답이 존재한다는 것을 알고 있을 때, 중간값을 기준으로 최적의 답을 찾기 위해 범위를 좁혀나가는 방식에 사용됩니다.
  2. 즉, 포인트 변수 lt와 rt 사이에 최적의 답(최대 랜선의 길이)가 존재하는 것을 알고 있기 때문에 lt와 rt의 중간값인 mid를 기준으로 범위를 좁힙니다.
  3. 리스트 변수 line의 첫 번째 요소 값을 기준으로 변수 mid의 값만큼 마구간 사이의 간격을 측정하여 마구간의 개수가 m보다 크거나 같은지를 확인합니다.
  4. 만약, count(mid) >= m인 경우라면 답이 될 수 있으므로 변수 res에 해당 mid 값을 할당합니다.
  5. 그리고 마구간 사이의 거리가 최대가 되는 최적의 답을 찾기 위해 변수 lt에 mid + 1을 할당하여 답의 범위를 좁힙니다.
  6. 반면, count(mid) < m인 경우라면 답이 될 수 없기 때문에 변수 rt에 mid - 1을 할당하여 새로운 답의 범위를 찾습니다.
  ```
