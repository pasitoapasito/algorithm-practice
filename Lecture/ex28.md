## Problem

> **문제**
  ```
  > A는 씨름 감독이다.
  > A는 씨름 선수를 선발하기 위해 공고를 냈으며, N명의 지원자가 지원을 했다. 
  > A는 각 지원자의 키와 몸무게 정보를 알고 있으며, A는 씨름 선수의 선발 원칙을 다음과 같이 정했다.
  
  > "다른 모든 지원자와 일대일 비교를 하여 키와 몸무게 중 적어도 하나는 크거나, 무거운 지원자만 뽑기로 했습니다."
  > 만약, B라는 지원자보다 키도 크고 몸무게도 무거운 지원자가 존재한다면 B지원자는 선발되지 않는다.
  > 위의 조건을 모두 만족시키는 프로그램을 작성하시오.

  - 입력설명
    * 첫 번째 줄에 지원자의 수 N(5 ≤ N ≤ 50)이 주어진다.
    * 두 번째 줄부터 N명의 키와 몸무게 정보가 차례로 주어진다.(단, 각 선수의 키와 몸무게는 모두 다름)

  - 출력설명
    * 첫 번째 줄에 씨름 선수로 뽑히는 최대 인원을 출력한다.
  ```

<br>
<hr>

## Solution

> **입력**
  ```
  5
  172 67
  183 65
  180 70
  170 72 
  181 60
  ```

<br>
<br>

> **풀이**
  ```python
  import sys
  sys.stdin=open('input.txt', 'rt')

  n = int(input())
  body = []
  for i in range(n):
      h, w = map(int, input().split())
      body.append((h, w))
  
  body.sort(reverse=True)
  largest = -2147000000
  cnt = 0
  for h, w in body:
      if w > largest:
          largest = w
          cnt += 1

  print(cnt)

  """
  출력: 3
  """
  ```

<br>
<br>

> **설명**
  ```
  1. sort(reverse=True) 메서드를 통해 튜플 형태의 원소를 가지는 리스트를 정렬시키면, 각 튜플의 첫 번째 원소 값을 기준으로 내림차순 정렬됩니다.
  2. 즉, 튜플 형태의 리스트 원소인 (h, w)에서 h를 기준으로 해당 리스트의 원소 값들이 우선적으로 내림차순 정렬됩니다.
  3. 만약 튜플의 첫 번째 원소인 h가 동일한 경우라면, 두 번째 원소인 w를 기준으로 내림차순 정렬됩니다.
  4. 이처럼, 입력값을 키순으로 내림차순 정렬시킨 이유는 키가 가장 큰 지원자는 반드시 선발되기 때문입니다.(키가 가장 크기 때문에 선발조건 충족함)
  5. 따라서, 우선적으로 키순으로 내림차순 정렬시킨 후에 몸무게를 기준으로 최대값이 경신되면 변수 cnt를 1만큼 증가시켜서 선발 인원을 카운팅합니다.
  6. 참고로, 몸무게의 최대값이 경신되었다는 것은 적어도 앞선 지원자보다 몸무게는 무겁다라는 것을 뜻합니다.(적어도 몸무게는 무겁기 때문에 선발조건 충족함)
  ```
