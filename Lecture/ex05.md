## Problem

> **문제**
  ```
  > N명 학생들의 수학점수가 주어집니다.
  > N명 학생들의 평균(소수 첫째자리에서 반올림)을 구하고, N명의 학생 중 평균에 가장 가까운 학생은 몇 번째 학생인지 출력하는 프로그램을 작성하시오.
  > 단, 평균과 가장 가까운 점수가 여러 개일 경우 점수가 가장 높은 학생의 번호(인덱스 번호)를 답으로 하고, 
  > 높은 점수를 가진 학생이 여러 명일 경우 (인덱스)번호가 가장 빠른 학생의 번호를 답으로 한다.

  - 입력설명
    * 첫 번째 줄에는 자연수 N(5 ≤ N ≤ 100)이 주어진다.
    * 두 번째 줄에는 각 학생의 수학점수인 N개의 자연수가 주어진다.
    * 참고로, 학생의 번호는 1부터 시작해서 N까지이다.
  
  - 출력설명
    * 첫 번째 줄에 평균과 평균에 가장 가까운 학생의 번호를 출력한다.
    * 참고로, 평균은 소수 첫째 자리에서 반올림한다.
  ```

<hr>

## Solution

> **입력**
  ```
  10
  45 73 66 87 92 67 75 79 75 80
  ```

<br>
<br>

> **풀이**
  ```python
  import sys
  sys.stdin=open('input.txt', 'rt')

  n = int(input())
  ch = list(map(int, input().split()))
  min = 2147000000

  avg = int(sum(ch)/n + 0.5)

  for idx, val in enumerate(ch):
      tmp = abs(val-avg)
      if tmp < min:
          min = tmp
          score = val
          res = idx + 1
      elif tmp == min:
          if val > score:
              score = val
              res = idx + 1

  print(avg, res)

  """
  출력: 74 7
  """
  ```

<br>
<br>

> **설명**
  ```
  1. 평균을 반올림 할 때, round() 메서드를 사용하지 않는 이유는 round() 메서드가 작동하는 방식이 round-half-even 방식이기 때문입니다.
  2. 즉, round() 메서드는 정확히 0.500을 만나면 짝수 방향으로 근사 값을 리턴하기 때문에 4.500의 경우 결과 값이 5가 아닌 4가 됩니다.
  3. 따라서 일반적인 반올림 방식(half-up)으로 평균값을 구하려면 평균값에 0.5를 더하고 int() 메서드를 활용해야 합니다.
  4. 만약, 동일한 점수일 경우 (인덱스)번호가 느린 학생의 점수를 구하려면 if문의 조건을 val >= score로 변경하면 됩니다.
  ```
