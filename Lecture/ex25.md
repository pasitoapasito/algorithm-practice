## Problem

> **문제**
  ```
  > A 레코드에서는 불세출의 가수 B의 라이브 동영상을 DVD로 만들어 판매하려고 한다.
  > DVD에는 총 N개의 곡이 들어가는데, DVD에 녹화할 때는 라이브에서의 순서가 그대로 유지되어야 한다.
  > 그 이유는 가수 B가 노래의 순서가 바뀌는 것을 매우 싫어하기 때문이다.
  > 즉, 1번 노래와 5번 노래를 같은 DVD에 녹화하기 위해서는 1번과 5번 사이의 모든 노래도 같은 DVD에 녹화해야 한다.
  > 또한, 한 노래를 쪼개서 두 개의 DVD에 녹화하면 안된다.

  > A 레코드 입장에서는 이 DVD가 팔릴 것인지 확신할 수 없기 때문에 이 사업에 낭비되는 DVD를 가급적 줄이려고 한다.
  > 고민 끝에 A 레코드는 M개의 DVD에 모든 동영상을 녹화하기로 하였다. 
  > 이 때 DVD의 크기(녹화 가능한 길이)를 최소로 하려고 한다.
  > 그리고 M개의 DVD는 모두 같은 크기여야 제조원가가 적게 들기 때문에 꼭 같은 크기로 해야 한다.
  > 위의 조건을 모두 만족시키는 프로그램을 작성하시오.

  - 입력설명
    * 첫 번째 줄에 자연수 N(1 ≤ N ≤ 1,000), M(1 ≤ M ≤ N)이 주어진다. 
    * 다음 줄에는 가수 B가 라이브에서 부른 순서대로 곡의 길이가 분 단위로 주어진다.
    * 단, 곡의 길이는 자연수이며 10,000분을 넘지 않는다.

  - 출력설명
    * 첫 번째 줄부터 DVD의 최소 용량 크기를 출력한다.
  ```

## Solution

> **입력**
  ```
  9 3
  1 2 3 4 5 6 7 8 9
  ```

<br>
<br>

> **풀이**
  ```python
  import sys
  sys.stdin=open('input.txt', 'rt')

  def count(capacity):
      cnt = 1
      sum = 0
      for x in music:
          if sum + x > capacity:
              cnt += 1
              sum = x
          else:
              sum += x
      return cnt

  if __name__ == '__main__':
      n, m = map(int, input().split())
      music = list(map(int, input().split()))
      maxx = max(music)

      lt = 1
      rt = sum(music)
      while lt <= rt:
          mid = (lt + rt) // 2
          if mid >= maxx and count(mid) <= m:
              res = mid
              rt = mid -1
          else:
              lt = mid + 1
      
      print(res)

  """
  출력: 17
  """
  ```

> **설명**
  ```
  1. 이분 검색은 특정 범위안에 답이 존재한다는 것을 알고 있을 때, 중간값을 기준으로 최적의 답을 찾기 위해 범위를 좁혀나가는 방식에 사용됩니다.
  2. 즉, 포인트 변수 lt와 rt 사이에 최적의 답(최대 랜선의 길이)가 존재하는 것을 알고 있기 때문에 lt와 rt의 중간값인 mid를 기준으로 범위를 좁힙니다.
  3. 변수 mid의 값(DVD의 녹화길이)가 용량이 가장 큰 음악보다 크고, count(mid) <= m인 경우라면 해당 값은 답이 될 수 있습니다.
  4. 따라서 해당 mid 값을 변수 res에 할당하고, 더욱 짧은 DVD 녹화길이를 찾기 위해 변수 rt에 mid - 1을 할당하여 답의 범위를 좁혀나갑니다.
  5. 반면에, count(mid) > m인 경우라면 답이 될 수 없으므로 변수 lt에 mid + 1을 할당하여 새로운 답의 범위를 찾습니다.
  6. 한편, DVD의 개수는 변수 capacity를 기준으로 음악 길이의 합인 sum과 비교하여 cnt의 개수를 증가시킵니다.
  7. 만약, 변수 sum에 새로운 음악을 더했을 때 capacity를 초과한다면 변수 cnt를 1만큼 증가시키고 sum은 새로운 음악(x)로 초기화합니다.
  ```
